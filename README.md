# Digda
Watches file changes and shows file space difference

## Usage
[Digda.exe path] [args]

### Args
-h: shows help <br>
-r: sets root directory as watching directory <br>
-u: updates file logs <br>

### In-program commands
[q | quit]: quits program <br>
[s | show]: write changes in file <br>
[h | help]: shows help <br>
